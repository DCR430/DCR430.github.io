Fresh Paws!

A Directory of 400+ NYC High Schools where users can publish and share reviews. Think of it this way, there are number of sites for reviewing restaurants but there aren't any apps that lets parents and students leave reviews on our public education system. See the problem here? Now you can read all about that school down the block before sending your kids there for the next four years. (yikes)

Tech Stack:

React, Redux, Ruby on Rails API, NYC Open Data API, Google Maps API, PostgreSQL, Active Record Rack Cors, Active Model Serializer, BCrypt, JWT, Semantic UI, HTML, CSS
