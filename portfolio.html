Projects



Fresh Paws

A Mobile Application in which a user can create an account and sign in to see a directory of local pet groomers in there area. A user can see services provided by the groomers, book appointments and leave reviews. During quarantine and grooming my own dogs, I realized how imporatnt finding a quality pet groomer was!

Tech Stack

React-Native, Redux, Ruby on Rails API, Google Maps API, PostgreSQL, Active Record, Rack Cors, Active Model Serializer, Google Firebase, HTML, CSS



Vibe Music

A Vanilla JavaScript application in which users can search for there favortie artists and add there favorite songs to a playlist. A user can search for an artist, once their favorite artist is found, the selected artist top 5 songs will show. A user can then listen to a preview of the song and add it to their "vibes playlist".

Tech Stack

Vanilla JavaScript, Ruby on Rails API, Deezer API, PostgreSQL, Active Record, Rack Cors, Active Model Serializer, HTML, CSS



Garden 2 Table

A web application in which users can search for vegetables, once a vegetable is found it can be added to the users virtual garden. Once the vegetable is added to the garden the user can query vegetable based recipes along with nutritional facts about the vegetable. Users are also able to like the recipes they have used.

Tech Stack

React, Ruby on Rails API, PostgreSQL, Active Record, Active Model Serializer, BCrypt, Semantic UI, HTML, CSS
